## 0.0.2

- Simplified color API: `colorRed`, `colorRGB` etc. apply foreground directly
- Added background color functions: `bgRed`, `bgRGB`, `bg256` etc.
- Raw color values available via `Color.red`, `Color.rgb` etc. for `marginColor`/`underlineColored`

- Initial release
- Boxes, tables, trees, lists, and horizontal rules
- Row-based layouts with alignment
- UTF-8 box-drawing support
- Zero dependencies
